POINTERS BASICS

This folder contains my first introduction to pointers in C.

The goal of this part is to understand how memory works and how to
manipulate values using addresses.

What I learned:

- A variable has a value and an address in memory
- '&' is used to get the address of a variable
- '*' is used to access or modify the value stored at an address
- Pointers allow modifying a variable directly from a function
- It is possible to have multiple levels of pointers (** , *** , etc)
- Pointers can be used to return multiple values from a function

Exercises done:

- ft_pointer_basic → understanding address and pointer
- ft_set → modify a value using a pointer
- ft_swap → swap two values using pointers
- ft_ultimate_ft → understand multiple levels of pointers
- ft_div_mod → return multiple values using pointers

Conclusion:

Pointers are essential in C because they allow direct access to memory
and make functions more powerful by modifying real values instead of copies.
