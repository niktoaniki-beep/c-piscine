# DAY 05 — REINFORCEMENT RESTART

This folder contains my return to C practice after almost one week away from coding.

Last week was difficult because I had a lot of pressure from the 1337 check-in.
I was stressed and slowly became disconnected from coding practice.

Even after getting my check-in successfully, I felt lost and confused when coming back.

So instead of rushing into new concepts, I decided to restart slowly with reinforcement exercises.

What I practiced today:

- ft_set → modifying a real value using a pointer
- ft_swap → swapping two variables using pointers
- ft_div_mod → returning multiple values with pointers
- ft_strlen → counting characters in a string
- ft_putstr → displaying a string using write

What I relearned:

- how pointers modify real memory values
- how strings work in C
- using loops with strings
- using write instead of printf
- how '\0' marks the end of a string

Conclusion:

Today was not about learning difficult things.
It was about reconnecting my brain to C programming after a break.

Even if I felt lost at the beginning, I realized that the logic was still there.
I just needed practice again.
