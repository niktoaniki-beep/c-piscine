# C Logic Training

This repository contains a collection of exercises I completed to strengthen my logical thinking in C.

## Focus Areas

* Loops (`while`)
* Conditions (`if`)
* Nested loops
* Patterns (triangles, grids, etc.)
* Output using `write`

## Learning Process

At the beginning, I struggled to translate logic into C code.
Through consistent daily practice, this process has become more natural and intuitive.

## Objective

My goal is to:

* Become fast and efficient when solving problems
* Translate ideas into clean and working C code
* Build strong problem-solving habits for future projects

## Long-Term Vision

Even as I move forward to more advanced topics such as:

* Arrays
* Strings
* Pointers

I will continue practicing these fundamental exercises to maintain and improve my core logic skills.

This repository represents the foundation of my C programming journey.
