# Arrays Logic Training (C)

This folder contains exercises focused on improving logical thinking in C using arrays.

## Topics Covered

* Counting occurrences of values
* Detecting duplicates and unique values
* Finding most frequent and least frequent elements
* Filtering arrays based on conditions
* Using nested loops for complex logic

## Objective

The main goal is to strengthen my ability to transform a problem into step-by-step C code.

I focus on:

* Breaking problems into smaller steps
* Writing clean and structured loops
* Avoiding logical errors (indexes, conditions, counters)

## Learning Progress

Some problems were challenging at first, especially:

* Most/least frequent values
* Nested loop logic
* Managing counters correctly

With consistent practice, I improved my ability to:

* Think algorithmically
* Debug logic mistakes
* Write more efficient solutions

## Future Goals

* Apply this logic to strings and pointers
* Optimize solutions (reduce complexity)
* Prepare for 1337 / 42 exam-style problems
