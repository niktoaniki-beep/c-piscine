# Strings Basics - C

This folder contains basic string manipulation exercises in C.

## Functions implemented

- ft_strlen      : calculate string length
- ft_putstr      : print a string using write
- ft_strcpy      : copy a string into another
- ft_strcmp      : compare two strings
- ft_str_is_alpha    : check if string contains only letters
- ft_str_is_numeric  : check if string contains only digits
- ft_str_is_lowercase: check if string contains only lowercase letters
- ft_str_is_uppercase: check if string contains only uppercase letters

## Concepts learned

- String traversal using while loops
- Character comparison using ASCII ranges
- Null terminator ('\\0')
- Conditions and logical operators (&&, ||, !)
