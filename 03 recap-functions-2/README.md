# Bonus Exercises - C Basics

While learning the basics of C programming, I realized that I didn't fully understand some concepts at first.

To strengthen my foundation, I created this folder with additional exercises focused on:

## Topics

- while loops
- if conditions
- using write()
- number manipulation (/10 and %10)
- working with characters

## Goal

The goal of these exercises is to:
- improve my understanding of basic logic
- become comfortable with writing functions
- build strong fundamentals before moving to more advanced topics

## Exercises included

- print numbers (0 to 9)
- reverse numbers (9 to 0)
- even and odd numbers
- print numbers from 00 to 99
- reverse printing (99 to 00)
- multiples filtering (e.g. multiples of 3)
- special conditions (replace values like multiples of 5)
- reverse number printing

---

This practice helped me understand core programming concepts more deeply and improve my problem-solving skills.
