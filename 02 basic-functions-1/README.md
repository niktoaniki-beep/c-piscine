# Basic Functions 1

This folder contains basic C exercises to build strong foundations.

## Topics covered

- Writing simple functions
- Using write()
- Working with integers
- Number manipulation (/10 and %10)
- Handling negative numbers
- Basic logic with loops and conditions

## Exercises

- ft_putchar
- ft_add
- ft_abs
- ft_last_digit
- ft_first_digit
- ft_reverse_number
- ft_print_to_98
...

Goal: understand how numbers work in C and improve problem solving.
