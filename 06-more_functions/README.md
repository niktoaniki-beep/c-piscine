# More Functions (C)

This folder contains additional C functions focused on character manipulation and basic logic.

## Topics covered

- Checking character types:
  - uppercase letters (`A-Z`)
  - lowercase letters (`a-z`)
  - digits (`0-9`)
  - alphabetic characters

- Character transformations:
  - converting uppercase to lowercase
  - converting lowercase to uppercase

## Functions implemented

- ft_isupper
- ft_isdigit
- ft_isalpha
- ft_tolower
- ft_toupper

## Notes

All functions are implemented using basic C logic without using standard library helpers.

The goal is to fully understand how characters work internally (ASCII values) and how to manipulate them manually.

This practice helps build a strong foundation for future low-level programming and system-level work.
